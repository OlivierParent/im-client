# X Reality

## Installation

    $ npm install

## Configuration

In the file `package.json`: change `"homepage": "/x-reality"` to the URL of your personal GitHub Pages.

## Develop

    $ npm start

## Build

    $ npm run build

## Known Issues

React Three Gui and React 17 don't play well together. Keep "react": "^16.14.0" and "react-three-gui": "0.3.1".
